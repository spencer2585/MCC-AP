# Setup Guide
> [!CAUTION]
> Due to the modding method this uses windows antivirus may flag the exe. If it does then often the game will severly lag and the exe will dissapear from the folder. If this is the case folow these steps to fix it
> 1. Open Windows Security -> Virus & threat protection
> 2. Under Virus & Threat protection click Manage Settings
> 3. scroll to excusions -> add or remove exclusions
> 4. add the folder containing HaloAP_Injector to the list
> You may need to restore the exe from quarantine if so
> 1. if Virus & Threat Protection -> Protection History
> 2. Find the Exe and restore it
1. Download the zip file from the github and extract to a folder of your choice.
2. Generate a game
3. Run HaloAP_Injector.exe
4. Enter your room info
5. Start MCC in Anti-Cheat Disabled mode. This will not work with the normal MCC Launch. if steam does not ask you how you would like to launch the game you can to to properties general and change your selected launch option.
6. Play the game
